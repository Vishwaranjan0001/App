# Zero Hunger Connect - Frontend (Vite)

This folder contains a minimal Vite + React scaffold with Login and Register pages for Donators and NGOs.

Quick start:

```bash
cd frontend
npm install
npm run dev
```

- Register a Donator or NGO via `/register` (stored in localStorage for demo).
- Login via `/login`.

This is a front-end scaffold only; to connect to your backend, update API calls accordingly.
